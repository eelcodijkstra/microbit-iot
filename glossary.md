# Glossary

```{glossary}
downlink-bericht
  Bericht naar een IoT-knoop, meestal voor het aansturen van actuatoren, of voor het instellen van de configuratie van de knoop. 

uplink-bericht
  Bericht van een IoT-knoop, meestal met sensor-waarden.

Cayenne Low Power Payload (LPP) formaat
  Binair formaat voor sensor- en actuatordata, o.a. gebruikt in LoRaWan netwerken. (In dit materiaal gebruiken wel ook een JSON-versie van dit formaat, voor de MQTT berichten.)

LPP
  Low Power Payload, zie Cayenne LPP

JSON
  JavaScript Object Notation, een manier om JavaScript objecten als string weer te geven, voor communicatie of opslag in een database. Dit formaat wordt ook door andere talen, zoals Python, ondersteund.
```
